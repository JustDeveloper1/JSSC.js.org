# Кодировки Символов в JSSC
JSSC использует свои **внутренние кодировки символов** как часть алгоритма сжатия.

Данные кодировки **не относятся** к ASCII, UTF-8, UTF-16, Windows-125x, ISO-8859 или к любым другим системным или платформенным кодировкам.

Они существуют **только внутри JSSC** и используются для оптимизации сжатия для многих языков (только для самых популярных).

Кодировки Символов используются [**Режимом Сжатия `05` — «Кодировка Символов»**](../modes/05.md).
Существует всего **15 Кодировок Символов** для **23 языков**.

## Список Кодировок Символов в JSSC
- [`00` **JSSCBASE**](base) — Нет языков - Пунктуация, числа, валюты и символы
- [`01` **JSSCRU**](ru) — Русский, Украинский, Белорусский, Казахский
- [`02` **JSSCENRU**](enru) — Английский, Русский, Украинский, Белорусский
- [`03` **JSSCENKK**](enkk) — Английский, Русский, Казахский
- [`04` **JSSCHI**](hi) — Хинди
- [`05` **JSSCENHI**](enhi) — Английский, Хинди
- [`06` **JSSCBN**](bn) — Бенгальский
- [`07` **JSSCENBN**](enbn) — Английский, Бенгальский
- [`08` **JSSCHIBN**](hibn) — Хинди, Бенгальский
- [`09` **JSSCJA**](ja) — Английский, Японский (Хирагана и Катакана)
- [`10` **JSSCTelu**](telu) — Английский, Телугу
- [`11` **JSSCMR**](mr) — Английский, Маратхи
- [`12` **JSSCB**](b) — *«Балтика»* - Латышский, Литовский, Эстонский, Польский, Чешский, Словацкий
- [`13` **JSSCE**](e) — *«Европейский»* - Английский, Французский, Немецкий, Испанский, Португальский, Итальянский, Нидерландский
- [`14` **JSSCAR**](ar) — Арабский
