# Режим Сжатия `05` — Character Encoding <Badge type="tip" text="с v1.0.0" />
Этот режим сжатия использует **[внутренние Кодировки Символов JSSC](../encodings/)**, чтобы сжать входные данные путём внутреннего перекодирования UTF-16 символов в символы самой оптимальной Кодировки Символов JSSC и выдаёт результат в виде UTF-16 символов.

Каждая Кодировка Символов JSSC содержит 256 символов, благодаря этому каждый символ помещается в 8 битов. Таким образом, два таких символа — это один UTF-16 символ.

## Пошаговый разбор
Входные данные: `Hello, World! Привет, Мир!` (Повторение «Привет, Мир» на английском и русском.)

1. Выбор самой оптимальной Кодировки Символов JSSC — в данном случае это [`JSSCENRU`](../encodings/enru).
2. Внутреннее перекодирование входных данных (далее представлены шестнадцатеричные коды):
   - `H` => `48`
   - `e` => `65`
   - `l` => `6C`
   - `l` => `6C` - **Обратите внимание, что JSSC input RLE может поставить `2` (`32`) здесь вместо `l` (`6C`).**
   - `o` => `6F`
   - `,` => `2C`
   - <code style="color:transparent!important;">_</code> => `20`
   - `W` => `57`
   - `o` => `6F`
   - `r` => `72`
   - `l` => `6C`
   - `d` => `64`
   - `!` => `21`
   - <code style="color:transparent!important;">_</code> => `20`
   - `П` => `8F`
   - `р` => `B0`
   - `и` => `A8`
   - `в` => `A2`
   - `е` => `A5`
   - `т` => `B2`
   - `,` => `2C`
   - <code style="color:transparent!important;">_</code> => `20`
   - `М` => `8C`
   - `и` => `A8`
   - `р` => `B0`
   - `!` => `21`
3. Выдать результат в виде UTF-16 символов:
   - `48` `65` => `{{ String.fromCharCode(0x4865) }}`
   - `6C` `6C` => `{{ String.fromCharCode(0x6C6C) }}`
   - `6F` `2C` => `{{ String.fromCharCode(0x6F2C) }}`
   - `20` `57` => `{{ String.fromCharCode(0x2057) }}`
   - `6F` `72` => `{{ String.fromCharCode(0x6F72) }}`
   - `6C` `64` => `{{ String.fromCharCode(0x6C64) }}`
   - `21` `20` => `{{ String.fromCharCode(0x2120) }}`
   - `8F` `B0` => `{{ String.fromCharCode(0x8FB0) }}`
   - `A8` `A2` => `{{ String.fromCharCode(0xA8A2) }}`
   - `A5` `B2` => `{{ String.fromCharCode(0xA5B2) }}`
   - `2C` `20` => `{{ String.fromCharCode(0x2C20) }}`
   - `8C` `A8` => `{{ String.fromCharCode(0x8CA8) }}`
   - `B0` `21` => `{{ String.fromCharCode(0xB021) }}`

Выходные данные: `{{ String.fromCharCode(0x4865) + String.fromCharCode(0x6C6C) + String.fromCharCode(0x6F2C) + String.fromCharCode(0x2057) +  String.fromCharCode(0x6F72) + String.fromCharCode(0x6C64) + String.fromCharCode(0x2120) + String.fromCharCode(0x8FB0) + String.fromCharCode(0xA8A2) + String.fromCharCode(0xA5B2) + String.fromCharCode(0x2C20) + String.fromCharCode(0x8CA8) + String.fromCharCode(0xB021) }}`
