# Compression Mode `05` — Character Encoding <Badge type="tip" text="since v1.0.0" />
This mode uses **[JSSC internal Character Encodings](../encodings/)** to compress the input string by internally remapping UTF-16 characters into the most optimal JSSC encoding and emitting the result back as UTF-16.

Each JSSC encoding contains 256 symbols, meaning each encoded symbol fits into 8 bits. One UTF-16 character (16 bits) is therefore represented by two JSSC-encoded symbols.

## Step-by-Step Walkthrough
Input: `Hello, World! Привет, Мир!` (It says "Hello, World!" in English and Russian.)

1. Select the most optimal JSSC Character Encoding: in this case, it would be [`JSSCENRU`](../encodings/enru).
2. Internally remap input characters to JSSC encoding (hex):
   - `H` → `48`
   - `e` → `65`
   - `l` → `6C`
   - `l` → `6C` - **Note that JSSC input RLE may put `2` (`32`) here instead of `l` (`6C`).**
   - `o` → `6F`
   - `,` → `2C`
   - <code style="color:transparent!important;">_</code> → `20`
   - `W` → `57`
   - `o` → `6F`
   - `r` → `72`
   - `l` → `6C`
   - `d` → `64`
   - `!` → `21`
   - <code style="color:transparent!important;">_</code> → `20`
   - `П` → `8F`
   - `р` → `B0`
   - `и` → `A8`
   - `в` → `A2`
   - `е` → `A5`
   - `т` → `B2`
   - `,` → `2C`
   - <code style="color:transparent!important;">_</code> → `20`
   - `М` → `8C`
   - `и` → `A8`
   - `р` → `B0`
   - `!` → `21`
3. Emit as UTF-16:
   - `48` `65` → `{{ String.fromCharCode(0x4865) }}`
   - `6C` `6C` → `{{ String.fromCharCode(0x6C6C) }}`
   - `6F` `2C` → `{{ String.fromCharCode(0x6F2C) }}`
   - `20` `57` → `{{ String.fromCharCode(0x2057) }}`
   - `6F` `72` → `{{ String.fromCharCode(0x6F72) }}`
   - `6C` `64` → `{{ String.fromCharCode(0x6C64) }}`
   - `21` `20` → `{{ String.fromCharCode(0x2120) }}`
   - `8F` `B0` → `{{ String.fromCharCode(0x8FB0) }}`
   - `A8` `A2` → `{{ String.fromCharCode(0xA8A2) }}`
   - `A5` `B2` → `{{ String.fromCharCode(0xA5B2) }}`
   - `2C` `20` → `{{ String.fromCharCode(0x2C20) }}`
   - `8C` `A8` → `{{ String.fromCharCode(0x8CA8) }}`
   - `B0` `21` → `{{ String.fromCharCode(0xB021) }}`

Output: `{{ String.fromCharCode(0x4865) + String.fromCharCode(0x6C6C) + String.fromCharCode(0x6F2C) + String.fromCharCode(0x2057) +  String.fromCharCode(0x6F72) + String.fromCharCode(0x6C64) + String.fromCharCode(0x2120) + String.fromCharCode(0x8FB0) + String.fromCharCode(0xA8A2) + String.fromCharCode(0xA5B2) + String.fromCharCode(0x2C20) + String.fromCharCode(0x8CA8) + String.fromCharCode(0xB021) }}`
